# Shrimkage

A CLI Tool for Image Optimizations
`shrink`+`image`

## Commands

### 0. help

```
shrimkage --help
```

### 1. file

optimizes an given image

```
shrimkage file --path=./example.png
```

#### Options

1. `--level`: default = 50%

### 2. folder

optimizes all the images in the folder

```
shrimkage folder
```

#### Options

1. `--level`: default = 50

> NOTE: Works only on PNG/JPG
